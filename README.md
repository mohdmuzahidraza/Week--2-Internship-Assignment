# Java OOP Assignments & Mini Project

This repository contains Java programs demonstrating fundamental Object-Oriented Programming (OOP) concepts such as Classes, Objects, Inheritance, Abstraction, Encapsulation, and Method Overriding.

- Method Overriding

# 3. Library Management System (Mini Project)

Description
A simple Library Management System built using Java OOP concepts. The application allows adding books, borrowing books, and returning books.

Features
- Add Books
- Borrow Books
- Return Books
- User Management
- Book Availability Tracking

OOP Concepts Demonstrated

Encapsulation
- Book details are stored using private variables and accessed through methods.

Inheritance
- `Stud` class inherits from the abstract `User` class.

Abstraction
- `User` is an abstract class.

Polymorphism
- Method overriding through `displayRole()`.

Classes Used
- Book
- Library
- User
- Stud

Sample Output
```
Book Added: Java Programming
Book Added: Data Structures
Muzahid is a Student.
Book Borrowed: Java Programming
Book Returned: Java Programming
```
